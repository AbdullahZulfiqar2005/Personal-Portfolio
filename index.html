Interactive Chess Game using JavaScript

This project is an interactive chess game implemented in JavaScript, designed to provide a dynamic and engaging experience for chess enthusiasts. It features a functional chessboard where players can move pieces, highlight legal moves, and detect check and checkmate conditions.

Key Features:

  • Interactive Board: A visually appealing chessboard with alternating colored squares and image assets for pieces.
  • Piece Movement: Dynamic piece selection and movement with highlighted legal moves.
  • Check and Checkmate Detection: Real-time detection of check and checkmate conditions with alerts.
  • Turn Management: Alternates between white and black players after each valid move.
  • Move Validation: Validates moves based on chess rules for different pieces, handles capturing and blocking moves.
  • Responsive UI: Ensures correct rendering of the chessboard and pieces across different screen sizes.

Technology used: HTML, CSS, JavaScript
